HTML Table Colspan & Rowspan

HTML tables can have cells that spans over multiple rows and/or columns.

HTML Table - Colspan

To make a cell span over multiple columns, use the colspan attribute:

Name Age
Jill Smith 43
Eve Jackson 57

HTML Table - Rowspan

To make a cell span over multiple rows, use the text-align attribute:

Name Jill
Phone 555-1234
555-8745
